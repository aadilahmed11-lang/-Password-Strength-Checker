PROJECT DESCRIPTION:
      This is a simple student project developed using Python. The project checks whether a password is weak, medium, or strong.
PURPOSE:
     The main purpose of this project is to help users create strong and secure passwords by giving suggestions for improvement.
TECHNOLOGIES USED:
     - Python
     - Tkinter
     - Regular Expressions
     - NLTK
HOW IT WORKS:
     1. The user enters a password.
     2. The program checks length, letters, numbers, and symbols.
     3. It also checks dictionary words.
     4. The result is shown as weak, medium, or strong.
OUTPUT:
     The program displays password strength and suggestions.
